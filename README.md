# IONIC-CLI ANGULAR
 ionic client use angular
 
# Core Setup
$ npm install -g ionic
$ ionic start name blank
$ cd name 
$ ionic serve

# Angular Setup
$ npm install @ionic/angular
$ npm run build
$ ng new name
$ cd name
$ npm install @angular-devkit/schematics-cli
$ npm link @ionic/angular

# Native Setup
$ npm install @ionic-native/core --save
